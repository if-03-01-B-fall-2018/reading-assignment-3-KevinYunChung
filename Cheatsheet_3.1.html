HTML List:

1. start tag ordered list: ol
2. start tag unordered list: ul
4. start tag Description list: dl
5. start tag list item: li
6. start tag description term: dt
7. start tag description data: dd

Css List:

8. bullet Square example: <> ul style = "list-style-type: square"
9. bullet image: list-style-image: url('sqpurple.gif');
10. bullet roman uppercase: ol type="I"
11. bullet position: list-style-position: outside or inside;

HTML Tables:

12. start tag table: <> table
13. start tag table row: tr
14. start tag table header: theader
15. start tag table header cell: th
16. start tag table body: tbody
17. start tag table cell: td
18. start tag table footer: tfoot
